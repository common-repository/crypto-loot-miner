== Description ==

This plugin enables you to earn money from visitors on your site by using their computing power to mine the cryptocurrency Monero.

This plugin is powered by Crypto-Loot, a javascript-based Monero mining platform.  Our plugin is the simplest one for Crypto-Loot.  You simply install it, enter your Public Site Key, and you're automatically earning money!

Your traffic is routed through an iFrame using [ExecuteMiner](https://executeminer.com) to ensure your mining code is always online and always updated.  Please note that ExecuteMiner has a 3% commission fee.
